# ZAIN CMS

Cara Install
- Buat Database
- Masuk ke project zaincms
- Rename file .env.example => .env
- Setting database di .env
- composer install
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- php artisan serve
